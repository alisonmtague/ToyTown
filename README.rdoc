== Toy Town

Toy Town is an app I created for a small business. It is a site that allows users to sign in if they wish to create wishlists and also have the capability to vote for their favorite popular toys. 

Technologies used:

* Ruby 2.1.2p95
* Postgresql
* OmniAuth for Twitter
* Acts as votable

User Stories
* User can access the site without sign in
* User can sign in with Twitter
    - User can create wishlists
    - User can vote on toys
* User can navigate pages
